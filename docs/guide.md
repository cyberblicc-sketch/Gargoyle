# Gargoyle – AI Shadow Consultant Guide

## Overview
Gargoyle is a multi‑agent AI platform that acts as your **shadow consultant** for regulatory, legal and strategic challenges.  It combines a **Jurist** for legal analysis, a **Cartographer** for patent & knowledge mapping, and a **Strategist** for adversarial simulations.  Leveraging the latest research in multi‑agent legal reasoning【139736145795859†L49-L64】 and robust agentic frameworks【985533924939773†L170-L199】, Gargoyle delivers reliable insights at a fraction of the cost of traditional legal services.

This package contains:

- **gargoyle_app/** – Front‑end website (responsive) and assets.
- **server/** – Python demo of the multi‑agent orchestrator.
- **docs/** – Business plan, roadmap, marketing materials, and grant resources.

## How to Run the Demo
1. Install Python 3.10+ and virtual environment tools.
2. Navigate to `server/` and install dependencies: `pip install -r requirements.txt`.
3. Run the orchestrator demo: `python gargoyle_orchestrator.py` and follow the prompt.
4. To view the front‑end, open `gargoyle_app/index.html` in a web browser.  Optionally host with `python -m http.server`.
